# Petit pendu
Petit jeu du pendu, multijoueur

## Installation

### Poetry
```commandline
poetry install
```
### Sinon
Par exemple dans un venv :
```commandline
pip install -r requirements.txt
```

## Lancement

### Poetry
Dans la racine du projet :
```commandline
poetry run python -u src/presentation/app.py
```
### Sinon
Par exemple dans un venv :
```commandline
python -u src/presentation/app.py
```
